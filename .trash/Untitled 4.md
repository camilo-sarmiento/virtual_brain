# Modos de animar


## Hacia Adelante
> *Liberar la creatividad*

Esta manera deja que la acción fluya, sin saber en donde va a terminar, solo se preocupa por el presente.

Esta es una buena técnica para poder descubrir cosas nuevas y sorpresas.

```ad-success
Las animaciones tendran mayor fluidez y naturalidad, además permite que la creatividad desarrolle cosas que no se tenian previstas y apoyen a la cción.

```

```ad-caution
Debe tener cuidado con por que si no se controla, la acción puede volverse esrratica, donde se pierde la uniformidad del personaje.

También, puede olvidarse el objetivo de la acción.

```

## BreackDown
> *Poses a Pose*
 
Estas son las poses secundarias que están entre las key poses, cuando tenemos claras nuestra poses que cuentan la historia por si mismas, debemos darle vida, para ello empezamos a insertar pose en medio para darle fluidez y naturalidad.

```ad-caution
Debe tener en cuenta los **Breackdowns**, por que puede tornar una acción **entrecortada** o muy **gomosa**.

```

## Method Blend
> *Una combinación del **BreackDown** y el **hacia delante** *

Este método mezcla la ventaja de ambos métodos, donde aprovechan de eso para tener las mejores animaciones.

<br/>

### Breackdown

Primero empezamos con los bocetos de las **key poses**, cuando estén bien definidas podemos pulir las proporciones o agregar alguna otra pose importante como anticipaciones o contacto de superficies.

<br/>

![[Pasted image 20221104122215.png]]

<br/>

### Hacia delante

Ya tenemos la parte de los **Breackdown**, ahora podemos usaremos la libertad del **hacia delante**, pero teniendo de guía la **key poses** para poder saber a donde nos dirigimos, esto nos da la oportunidad de improvisar.
<br/>

![[Pasted image 20221104122304.png]]

<br/>

### Repasos

Cuando tengamos terminemos, podremos hacer varios repasos, donde tendremos la oportunidad de corregir o cambiar las **key poses** y los **extremos**, y por ultimo agregar las acciones secundarias, como el pelo, ropa, cinturones, etc.
<br/>

![[Pasted image 20221104122320.png]]

```ad-success
Este método es una combinación entre estructura y fluidez.

Es un equilibrio entre planificación y espontaneidad.

```
